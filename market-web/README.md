## 简单的modal demo项目 modal-demo

### 技术栈
1. 使用Vue v.2.6
2. webpack 4.x
3. node v.12.x

### 项目运行
1. cnpm install 安装项目依赖
2. npm run dev 项目开发运行
3. npm run build 项目打包

### 项目结构
```
|—— simple-demo
    |—— public
        |—— index.html
        |—— assets
            |—— favicon.ico
            |—— global.js                           // 全局通用方法/配置
    |—— src
        |—— main.js
```
